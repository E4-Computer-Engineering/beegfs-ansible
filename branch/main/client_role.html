e4.beegfs.client role – Deploy Beegfs client services

Note

This role is part of the e4.beegfs collection (version 1.0.0).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it use: ansible-galaxy collection install e4.beegfs.

To use it in a playbook, specify: e4.beegfs.client.

Entry point main – Deploy Beegfs client services

New in e4.beegfs 1.0.0

Synopsis

  • Install and configure Beegfs client services for single and multi cluster environments.
